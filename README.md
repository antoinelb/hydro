# Hydro

Ce projet cherche être une preuve de concept de toute la chaîne de modélisation dans le but d'apprendre et démontrer l'hydrologie opérationnelle.

## Étapes de la chaîne de modélisation

### Modèle

1. Choix de la station hydrométrique (limitée aux stations hydrométriques du Québec mesurant le débit et avec des données remontant à au moins 10 ans)
2. Calibration du modèle sur les données avant les 5 dernières années
3. Validation du modèle sur les données des 5 dernières années

### Prévision

Mise en place d'un système de prévision

### Projection

1. Modèle climatique global
2. Modèle climatique local
3. Post-traitement statistique
4. Prévision et projection
